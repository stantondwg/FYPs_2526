# FYPs_2526 — Final Year Projects (2025–2026) — HPC & Linux Guidance

This repository contains short guides and example scripts to help Final Year Project students get started with High Performance Computing (HPC) and Linux on the university servers for the 2025–2026 academic year.

Contents
- 01 - Introduction to High Performace Computing and Linux.md  
  A brief introduction to HPC concepts and the basics of Linux relevant to running project workloads.
- 02 - Accessing the servers.md  
  Instructions for gaining access to the university compute servers, logging in, and basic account/security notes.
- 03 - Finding your way around gomphus in Linux.md  
  A practical walkthrough for navigating the gomphus system, paths, file locations and useful commands.
- 04 - Editing and submitting scripts.md  
  Guidance on editing job scripts, recommended editors and software, and how to submit jobs to the scheduler.

How to use these materials
- Read the guides in order if you're new to Linux and HPC.
- Copy or adapt the example scripts in your own project directories — do not edit shared files in the repo directly unless instructed.
- If you need to run code on the servers, follow the access and submission instructions in the relevant guides.
